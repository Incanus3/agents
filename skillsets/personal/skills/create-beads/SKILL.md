---
name: create-beads
description: Create beads for a specified task or the work discussed in the current session. Manual invocation only: apply only if the user explicitly invokes `$create-beads`; never select automatically from a natural-language request.
---

Create beads either for the work specified by the user or, when none is specified, for the work
discussed in the current session. Use the relevant skills. Make the beads reference source documents
so an agent assigned to a bead has all relevant context.
